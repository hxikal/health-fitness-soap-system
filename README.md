# 🏥 Personal Health & Fitness Calculator (SOAP Web Service)

## 📌 Project Overview

This project is a SOAP-based web service system developed to calculate various health metrics. The system allows users to input personal and lifestyle data and receive calculated health results.

## 🎯 Objectives

* To develop a SOAP-based web service
* To calculate health-related metrics such as BMI, body fat, and calories
* To implement additional lifestyle analysis modules

## ⚙️ Technologies Used

* Java (NetBeans)
* GlassFish Server
* SOAP Web Services
* HTML & PHP (Web Client)
* GitHub (Version Control)

## 🧩 Modules

### 🔹 Core Modules

* BMI Calculation
* Body Fat Percentage
* Calories Burn Rate
* Age Calculation

### 🔹 Additional Modules

* Sleep Quality Score
* Stress Level Estimator

## 🔄 System Flow

User → Web Client → SOAP Web Service → Process → Result

## 📁 Project Structure

* `backend/` → SOAP Web Service (NetBeans)
* `client/` → Web Client (HTML/PHP)
* `docs/` → Report and Presentation
